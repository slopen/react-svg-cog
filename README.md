# react-svg-cog


### install

```
$ git clone git@github.com:slopen/react-svg-cog.git
$ cd react-svg-cog
$ yarn
```

### build

```
$ yarn run build
```

### development

```
$ yarn run dev
```

* open `http://localhost:9041`
