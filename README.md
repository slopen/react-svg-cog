# react-svg-cog


### install

```
$ git clone git@github.com:slopen/react-svg-cog.git
$ cd react-svg-cog
$ yarn
```

### build

```
$ yarn run build
```

### development

```
$ yarn run dev
```

* open `http://localhost:9041`

### params

```
const props = {
  "color": "#000",
  "arcRadius": "0.238",
  "arcStrokeWidth": "0.032",
  "dentsTotal": "11",
  "dentsStrokeWidth": "0.751",
  "dentsStrokeMargin": "0.051"
};

```
